## ユニットコンテキスト

- **Should 落とし**: 要注意フラグ（US-A-012）・アラート履歴（US-A-022）は MVP スコープ外
- **感情タグ方式**: Unit-03 で感情スコア（1-5）を廃止し category (positive/negative/neutral) に変更済み
- **管理者のデータアクセス境界**: 集計データのみ閲覧可。日誌本文は一切返却しない
- **新規テーブル**: alerts（watch_flags は Should 落としで不要に）
- **新規 API**: 管理者用エンドポイント（school_admin のみ）
- **バッチ処理**: アラート自動検知（EventBridge + API Route or Lambda）
